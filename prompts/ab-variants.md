# A/B Variant Generator (Landing Page)

## Input
Paste the revised version (headline/subheadline/CTA/proof).

## Prompt
Role: You are a CRO strategist.
Task: Create 2 testable variants without changing layout.

Variant A — Risk Reduction
- Goal: reduce uncertainty, increase trust
Variant B — Outcome/ROI
- Goal: emphasize value/time/money impact

For EACH variant output:
- Headline
- Subheadline
- Primary CTA
- Hypothesis (If we do X, then Y will improve because Z)
- Primary metric + secondary metric
- Target audience note (who it resonates with)
