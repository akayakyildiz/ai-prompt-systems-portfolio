# CRO Evaluation Prompt (B2B SaaS)

## Input
Paste the landing page copy (headline → CTA → key sections).

## Prompt
Role: You are a CRO specialist for B2B SaaS landing pages.
Task: Evaluate the copy with a strict conversion-first mindset.

Score each criterion 1–10:
1) Value proposition clarity (first 5 seconds)
2) Proof & trust (logos, numbers, testimonials, credibility cues)
3) Objection handling (risk, switching cost, setup time)
4) CTA clarity & friction (next step, effort, commitment)

Output format:
- [Score Table] Criterion | Score | Evidence from copy
- [Top 3 Conversion Blockers] blocker → why → where it appears
- [Fix Plan] 5 prioritized changes (highest impact first)
- [Quick Wins] 3 edits that take <10 minutes
