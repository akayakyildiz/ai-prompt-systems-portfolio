# Controlled Landing Page Rewrite Prompt

## Inputs
- Persona + job-to-be-done
- Primary CTA goal (trial/demo)
- Top 3 objections
- Existing copy (paste)

## Prompt
Role: You are a conversion copywriter.
Task: Rewrite ONLY these parts while preserving structure:
- Headline
- Subheadline
- Primary CTA text + microcopy (what happens next)
- Proof section (use placeholders if real proof is missing)

Constraints:
- No exaggerated claims
- Keep tone: professional but friendly
- Do not add new features not mentioned
- Keep it concise; avoid jargon

Output format:
1) Revised headline + subheadline (2 options)
2) CTA block (primary + secondary + microcopy)
3) Proof block (placeholders + what to collect)
4) Rationale (bullet list: change → reason → expected impact)
